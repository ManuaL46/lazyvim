# LAZYVIM DOCS
My personal starter template for [LazyVim](https://github.com/LazyVim/LazyVim).
Refer to the [documentation](https://lazyvim.github.io/installation) to get started.

# PREREQUISITES
To use this seamlessly, install the following mandatory tools
* [Git](https://git-scm.com/install) - My preferred SCM
* [LazyGit](https://github.com/jesseduffield/lazygit.git) - Easy to use interface for git
* [Fzf](https://github.com/junegunn/fzf) - Fuzzy finder
* [Ripgrep](https://github.com/BurntSushi/ripgrep) - Alternative to grep

# INSTALLATION
## Linux
`mkdir -p ${HOME}/.config/nvim`
`cd ${HOME}/.config/nvim`
`git clone https://github.com/ManuaL46/lazyvim.git .`

## Windows
`mkdir %LOCALAPPDATA%\nvim`
`cd /d %LOCALAPPDATA%\nvim`
`git clone https://github.com/ManuaL46/lazyvim.git .`
